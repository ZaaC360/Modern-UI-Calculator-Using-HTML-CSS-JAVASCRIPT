# Animated Calculator

A modern and interactive calculator built using **HTML, CSS, and JavaScript**.  
This project focuses on DOM manipulation, UI animations, and dynamic user interaction.

# Features

- Basic arithmetic operations
- Smooth number animation from button to display
- Dark / Light theme toggle
- Calculation history panel
- Popup history viewer with close button
- Responsive calculator layout
- Clean and modern UI design

# Technologies Used

- HTML5
- CSS3 (Flexbox & Positioning)
- JavaScript (DOM Manipulation & Events)
- Local Storage

# Learning Objectives

This project helped practice:
- Event handling
- Dynamic element creation
- Absolute & relative positioning
- UI state management
- Animation using JavaScript
